# Automatic Grading System

This project is a Machine Learning and Natural Language Processing (NLP) based Automatic Grading System designed to evaluate descriptive answers automatically. The system compares a student's answer with the answer key using a BERT-based Sentence Transformer model and calculates semantic similarity using Cosine Similarity.

Based on the similarity score, the system generates marks and assigns grades automatically. A Streamlit-based dashboard provides an easy-to-use interface for displaying results.

## Features
- Automated descriptive answer evaluation
- BERT-based semantic analysis
- Cosine Similarity scoring
- Automatic marks and grade generation
- Streamlit interactive dashboard
- Result storage and display
